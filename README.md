A C# wrapper for VoiceIt's API 3.0 featuring Voice + Face Verification and Identification.

## Installation

```bash
git clone https://github.com/voiceittech/VoiceIt3-C-Sharp.git
```

Add the project as a reference in your solution.

## Getting Started

Sign up at [voiceit.io/pricing](https://voiceit.io/pricing) to get your API Key and Token, then log in to the [Dashboard](https://dashboard.voiceit.io) to manage your account.

<img src="./Graphics/getcredentials.png" alt="API Key and Token" width="400px" />

## API calls
You can visit our [HTTP API 3.0 Documentation](https://api.voiceit.io/?csharp#introduction) for detailed information on each API call.
